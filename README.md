# 🏦 Bank Management System (C++)

A console-based Bank Management System built using **C++** and **SQLite**.

## Features

- Create Account
- View Account
- Deposit Money
- Withdraw Money
- Update Account
- Delete Account
- SQLite Database Integration

## Technologies Used

- C++17
- SQLite3
- Makefile
- MinGW (Windows)

## Project Structure

```
BankManagementSystem/
│
├── include/
├── src/
├── libs/
├── database/
├── build/
├── Makefile
└── README.md
```

## Build

```bash
mingw32-make
```

## Run

```bash
mingw32-make run
```

## Clean

```bash
mingw32-make clean
```
